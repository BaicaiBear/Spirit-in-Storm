### 获得灵魂的途径
1. ✅食物（灵魂总量：[原版]41 [农夫乐事]～75）
   - 食用一种新的手持食物：+1
   - 食用一种新的放置食物：+1
2. ✅进度（灵魂总量：[原版]134）<br>_（目前为仅有原版进度可以）_
   - 完成一个新的普通进度：+0
   - 完成一个新的目标进度：+1
   - 完成一个新的挑战进度：+5
3. ✅群系探索 （灵魂总量：[原版]64）
    - 到达一种新的群系：+1
4. ✅特殊彩蛋
    - 将（金）胡萝卜重命名为■■■■：+1
    - 其他通过数据包实现的彩蛋：+1 <br> `通过检测namespace=spirit-in-storm, path.contain("egg/)的进度达成触发`
